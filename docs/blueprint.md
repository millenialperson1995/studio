# **App Name**: Retífica Ágil

## Core Features:

- Dashboard Metrics: Display key metrics (monthly/annual revenue, order statuses, pending quotes, customer metrics).
- Customer Management: CRUD operations for customer data (name, contact details, vehicles, service history).
- Vehicle Management: Manage vehicle details linked to customers (license plate, make, model, year, technical info, service history).
- Quotation System: Create, manage, and generate PDF quotations with itemized services/parts, totals, and customer/vehicle information.
- Service Order Management: Manage service orders, track status (pending, in progress, completed, canceled), assign mechanics, and record services/parts used.
- Inventory Management: Track parts inventory with stock levels, minimum quantities, purchase/sale prices, and supplier information.
- Service Management: Define and manage services offered with codes, descriptions, standard prices, average times, and categories.

## Style Guidelines:

- Primary color: Deep blue (#1A237E) to convey trust and professionalism, reflecting the precision required in vehicle repair.
- Background color: Light blue-gray (#ECEFF1), provides a clean, neutral backdrop that keeps attention on data.
- Accent color: Bright orange (#FF7043) to highlight important actions or notifications related to order status, acting as a clear call to action.
- Body and headline font: 'Inter', a grotesque-style sans-serif, known for its legibility on screens.
- Use clear, concise icons for navigation and status indicators, ensuring intuitive use in a mobile-first design.
- Mobile-first, prioritizing a responsive design with collapsible sections and clear hierarchies, and optimal viewing on small screens.
- Subtle transitions for screen changes and status updates to give users clear, quiet feedback when changes are applied in the app.
